# AVI проигрыватель для дисков «Игромании»

POC/MVP браузерного плеера для просмотра AVI-видео с дисков «Игромании» без установки старых кодеков в систему.

Он открывает `.avi` прямо в браузере, декодируя типичные архивные ролики DivX 5 / MPEG-4 Part 2 с MP3-звуком через собранный под WebAssembly `libav.js`.

Это часть разрабатываемого проекта, который позволит открывать и просматривать содержимое дисков «Игромании» в браузере.

## Запуск

Код не обязательно клонировать: плеер работает как статическая страница и может запускаться прямо с GitHub Pages.

На Windows:

```cmd
server.cmd
```

Скрипт поднимет локальный HTTP-сервер на `http://localhost:8000/` и откроет страницу в браузере.

Если нужен ручной запуск:

```sh
python -m http.server 8000
```

После этого откройте `http://localhost:8000/`.

## Использование

1. Нажмите `Open .avi file...` или перетащите AVI-файл на область плеера.
2. Управляйте воспроизведением кнопками внизу страницы.
3. Горячие клавиши: `Space` - пауза/воспроизведение, стрелки влево/вправо - перемотка, `F` - полноэкранный режим.

## Ограничения

Плеер рассчитан на AVI-файлы с MPEG-4 Part 2 / DivX-видео и MP3-аудио. Другие старые кодеки из AVI-контейнеров могут не открыться, если их нет в текущей сборке декодера.
